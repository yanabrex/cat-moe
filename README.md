# My Cat Moe
This is a simple HTML/CSS project showcasing my cat, Moe.
The page includes:

- About Moe
- His Favorite Things
- Fun Facts
- Some Quirks
- A small cat gallery
- A footer with a quote

This project was created to practice semantic HTML, CSS flexbox, responsive design, and basic web page structure.
